# npm

## regist user

```
$ npm set init.author.name "Your Name"
$ npm set init.author.email "you@example.com"
$ npm set init.author.url "http://yourblog.com"

$ npm adduser
```

## publish

```
$ npm init
$ npm publish ./
```

## update

```
$ git checkout v1.0.7
$ npm publish ./
$ npm dist-tag add foobar@1.0.7 latest
```
